🔥 Firebase 配置指引

請將下方 firebaseConfig 物件中的值替換為您 Firebase 專案的實際資料，並啟用 Firestore 資料庫。

// 在 <script> 區域取代原本的 firebaseConfig
const firebaseConfig = {
  apiKey: "請輸入您的 API Key",
  authDomain: "您的專案.firebaseapp.com",
  projectId: "您的專案ID",
  storageBucket: "您的專案.appspot.com",
  messagingSenderId: "寄件者ID",
  appId: "應用程式ID"
};

⚠️ 提醒：Firestore 規則需允許讀寫 (測試階段可開放)。投票記錄儲存於集合 votes 中，文件 ID 格式為「班級_學號」。

⚠️ 請先設定 Firebase 專案（上方管理工具內有配置代碼）並確保已啟用 Firestore。投票數據每人限投一次。
